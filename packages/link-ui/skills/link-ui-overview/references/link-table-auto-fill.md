# auto-table列属性auto-fill介绍

表格提供了一系列的属性来控制列宽（比如width、fit等），但在实际开发中，对这些属性的使用可能不规范

随意写的宽度可能会导致我们的列宽没有充分利用或者是浪费了很多空间，设计了一批规则(ui根据一些个性化字段设计了标准的尺寸)

下面是规则属性以及规则的相关描述，可以根据需求来使用对应的规则

**建议表格的每一列都使用auto-fill属性**

特殊场景: 部分客户可能希望**数字列靠左展示**, 此时由于 autoFill 覆盖了 align, 不能单独设置文字对齐, 所以需要去掉 autoFill 手动设置 `:width="120" align="left"` 或者设置为其他合适的规则

## autoFill、width、fit属性优先级

同时配置 autoFill、width、fit 属性，优先级排序 fit > autoFill > width
